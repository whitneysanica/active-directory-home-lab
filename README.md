# Active Directory Home Lab

## Overview

This project demonstrates the deployment of a Windows enterprise environment using Active Directory Domain Services (AD DS).

The lab was built entirely in Oracle VirtualBox and simulates a small business network.

---

## Environment

| Machine | Role |
|---------|------|
| DC01 | Windows Server 2022 Domain Controller |
| WIN10-01 | Windows 10 Enterprise Workstation |
